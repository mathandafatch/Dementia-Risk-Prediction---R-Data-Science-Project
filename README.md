# Dementia-Risk-Prediction---R-Data-Science-Project
Healthcare Dementia Prediction Data Analysis & Machine Learning Algorithms Implementation- Cleaned and analysed a medical dataset to explore relationships between BMI, nutrition and cognitive health. Performed feature selection and visualisation for model preparation.  Cognitive Decline Prediction Model (Machine Learning- Classification) Built and evaluated ML models (Logistic Regression, Decision Tree) to predict dementia risk factors. Prepared datasets, encoded features, split data, trained models and achieved strong predictive performance.
